Bien Démarer:

1. Importer la librairie

<script src="https://8borane8.github.io/updatejs/src/updatejs.js" type="application/javascript"></script>

Ajouter cette ligne de code avant votre script

2. Commencer à coder

js/index.js

Créer un fichier "index.js" dans un sous répertoire "js"

Exemples

<!DOCTYPE html>
<html lang="fr">
    <head>
        <title></title>
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <meta charset="utf-8">
                    
        <link rel="stylesheet" type="text/css" href="css/reset.css">
        <link rel="stylesheet" type="text/css" href="css/index.css">
    </head>
                    
    <body>
            
        <script src="../../src/updatejs.js" type="application/javascript"></script>
        <script src="js/index.js" type="application/javascript"></script>
    </body>
</html>
